# CanvasNote 操作マニュアル

## 1. 資料を始める

ダッシュボードの「新しい資料」で白紙を作成します。「読み込み」からPDF、HTML、または `.canvasnote.json` を開けます。

PDFは文字を編集要素へ変換します。スキャンPDFでは日本語OCRを使用します。複雑な図表は見た目を維持する固定画像レイヤーとして残ります。

## 2. 編集する

- キャンバス上の要素をクリックして選択します。
- 何もない場所からドラッグすると、範囲内の文字や図形をまとめて選択できます。
- `Shift + クリック` で選択を追加できます。
- 選択した要素をドラッグ、または矢印キーで移動します。`Shift + 矢印` は10px移動です。
- 右側の「選択項目をまとめて下へ」で、全体の配置を10px下へ修正できます。
- 右側から整列、均等配置、グループ化、複製、削除を実行できます。

## 3. 保存する

編集内容はこのPCのブラウザ内へ自動保存されます。別ブラウザや別PCへ移す場合は「書き出し」から「編集データ」を保存してください。

## 4. 書き出す

- PDF: 全ページを1つのPDFとして保存
- HTML: ブラウザで開ける編集情報付きHTML
- PNG: 現在のページを高解像度画像として保存
- 編集データ: CanvasNoteで再編集できるJSON

## 注意事項

OCR結果や特殊フォントは完全一致しない場合があります。読み込み後に文字を確認してください。80MBを超えるPDFは分割してから読み込んでください。
